# MS-Excel-dashboard-projects
📊 Netflix Content Analysis Dashboard | Excel Project

I recently built an interactive dashboard in Excel to analyze Netflix Movies & TV Shows data and uncover trends in content production, genres, and ratings.

🔍 Key Insights:
• Movies dominate the Netflix catalog compared to TV Shows
• The United States produces the highest amount of content
• Drama and International Movies appear as the most common genres
• Content production increased significantly after 2016.

🛠 Tools & Skills Used:
Excel | Pivot Tables | Pivot Charts | Data Cleaning | Data Visualization | Dashboard Design | Slicers
